| POST | `/api/v2/invites/[token]` | Accepts an invite, registers or updates the user, and maps them to the program. |
| DELETE | `/api/v2/kpis` | Deletes a KPI by id. |
| GET | `/api/v2/kpis` | Retrieves KPIs for a given program. |
| POST | `/api/v2/kpis` | Creates a new KPI with title and target value. |
| DELETE | `/api/v2/program-staff` | Removes a staff assignment by id. |
| GET | `/api/v2/program-staff` | Retrieves staff assignments based on staff_id or program_id. |
| POST | `/api/v2/program-staff` | Adds a staff assignment linking a staff member to a program. |
| GET | `/api/v2/teacher/fulfillment` | Fetches fulfillment data (participants, requirements, submissions) for a given program and week. |
| GET | `/api/v2/teacher/full-state` | Fetches full teacher state (programs, teams, submissions, sessions) for a given CID. |
| GET | `/api/v2/teacher/reports` | Fetches weekly reports without authentication. |
| POST | `/api/v2/teacher/reports` | Creates or updates a weekly report without authentication. |
