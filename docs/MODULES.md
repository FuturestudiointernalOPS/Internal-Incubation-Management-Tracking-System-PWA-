# Modules Reference

`src/lib`, `src/components`, `src/utils` — what each file exports and why it exists. Generated by reading each file (DeepSeek v4-flash via aider, reviewed by Claude).

## components/dashboard

| File | Exports | Purpose |
|---|---|---|
| `src/components/dashboard/AssignmentsView.js` | AssignmentsView | Provides an interface for participants to view and submit assignments, with filtering by program and status, and a submission modal for uploading files or URLs. |
| `src/components/dashboard/ParticipantDashboardHome.js` | ParticipantDashboardHome | Main participant dashboard showing welcome area, progress overview cards, calendar events, overdue/due-soon action center, and announcements. It serves as the landing page for participants. |
| `src/components/dashboard/ProfileView.js` | ProfileView | User profile page displaying personal information, enrolled programs, startup/group profile, goals, and recent submissions. Allows editing name and saving to API. |
| `src/components/dashboard/ProgramDetail.js` | ProgramDetail | Shows detailed view of a single program including curriculum weeks, deliverables, resources, progress metrics, submission history, follow-ups, and tabs for different sections. Used when participant clicks on a program card. |
| `src/components/dashboard/ProgramListing.js` | ProgramListing | Displays a list of programs for a participant with status badges, mini metrics, and a detailed card view. It fetches participant programs via API and groups them into active and other programs. |
| `src/components/dashboard/ProgressView.js` | ProgressView | Participant progress overview showing metrics (completion, attendance, assignments, KPIs, ritual participation), milestones, weekly breakdown history, and program-level selector. |
| `src/components/dashboard/RitualsView.js` | RitualsView | Manages participant rituals (standup, check-in, reflection) with form submission and history display. It fetches ritual data from API and shows recent activity. |
| `src/components/dashboard/UnifiedDashboard.js` | UnifiedDashboard | Aggregated dashboard for program managers and above, with unified calendar, assigned tasks, attention-required section, and consolidated workspace for programs, tasks, blockers, projects, and recent activity. |
